# 📈 Chartix - AI-Powered Crypto & Forex Platform

Chartix is a modern, comprehensive dashboard for Crypto and Forex traders, heavily integrated with Artificial Intelligence. It provides real-time market data, technical analysis, automated trend predictions using Machine Learning, and an intelligent RAG-based Chatbot powered by Llama 3 to assist with market news and insights.

## Core Features

- **AI Market Assistant (RAG)**: A dedicated AI assistant running on Groq (Llama 3). It fetches the latest news from ChromaDB (Vector DB) and provides well-reasoned trading suggestions and news summaries.
- **Trend Prediction (ML)**: Uses Machine Learning (XGBoost) trained on technical indicators (RSI, MACD, Volume) to predict market trends (Bullish/Bearish) for the next 4-8 hours.
- **Real-time Interactive Charts**: Powered by TradingView Lightweight Charts, plotting live OHLCV data directly from Binance WebSockets.
- **News & Economic Calendar Aggregator**: An automated NestJS background job that pulls, filters, and standardizes RSS feeds and news APIs to deliver the most impactful macro events.

## Architecture

Chartix is built as a **Monorepo** comprising three microservices:

1. **Frontend**: React + Vite + TypeScript. Professional Dark Mode UI tailored for traders.
2. **Backend**: NestJS + PostgreSQL + Redis. Handles authentication, API rate-limiting, and cronjobs for news crawling.
3. **AI Service**: Python + FastAPI. Houses the RAG pipeline (sentence-transformers + ChromaDB) and ML inference models.

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python 3.11+
- Docker & Docker Compose

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/chartix.git
   cd chartix
   ```

2. **Start Infrastructure (Databases & Cache)**
   ```bash
   docker-compose up -d
   ```

3. **Install Dependencies**
   - **Frontend**: `cd frontend && npm install`
   - **Backend**: `cd backend && npm install`
   - **AI Service**: `cd ai-service && pip install -r requirements.txt`

4. **Environment Variables**
   - Create `.env` files in each respective directory based on their `.env.example` templates.
   - *Note: You will need a free Groq API key for the AI features.*

5. **Run the Services**
   - Frontend: `npm run dev` (Runs on `http://localhost:5173`)
   - Backend: `npm run start:dev` (Runs on `http://localhost:3000`)
   - AI Service: `cd ai-service && uvicorn app.main:app --reload` (Runs on `http://localhost:8000`)
